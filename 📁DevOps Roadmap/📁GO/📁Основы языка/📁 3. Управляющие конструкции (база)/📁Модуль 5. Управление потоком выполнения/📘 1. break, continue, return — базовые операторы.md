
## Описание

Эти три ключевых слова позволяют управлять потоком выполнения внутри циклов и функций. Они фундаментальны для построения логики.

---

## `break`

Прерывает выполнение ближайшего внешнего цикла или `switch`.

```go
for i := 0; i < 10; i++ {
    if i == 5 {
        break
    }
    fmt.Println(i)
}
```

---

## `continue`

Прерывает текущую итерацию и переходит к следующей. Тело цикла ниже `continue` не выполняется.

```go
for i := 1; i <= 5; i++ {
    if i%2 == 0 {
        continue
    }
    fmt.Println(i) // только нечётные
}
```

---

## `return`

Завершает выполнение функции. Может возвращать значения, если они указаны в сигнатуре функции.

```go
func greet(name string) string {
    if name == "" {
        return "Привет, гость"
    }
    return "Привет, " + name
}
```

---

## Совместное использование

Можно комбинировать `break` и `continue` в условиях:

```go
for i := 1; i <= 10; i++ {
    if i%3 == 0 {
        continue
    }
    if i > 7 {
        break
    }
    fmt.Println(i)
}
```

---

## Вложенность

Все три оператора действуют только в рамках **текущей** области: цикла или функции.

---

## Практика

1. Напиши цикл от 1 до 20, который завершает выполнение, дойдя до числа, делящегося на 9.
2. Используй `continue`, чтобы пропускать числа, делящиеся на 4.
3. Напиши функцию, которая возвращает результат "OK", если вход > 10, иначе — "FAIL" (с использованием `return`).
4. Реализуй функцию, которая ищет число в массиве. Если найдено — выйти из цикла (`break`) и вернуть индекс (`return`). Если нет — вернуть -1.
